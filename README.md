# AutoriDeobfuscator
.NET Deobfuscator for AutoriObfuscator
![alt text](prev.png)

# Features
it removes:
- Numeric Encryption
- String Encryption
- Constant Melter
- Constructor Decomp
- Empty Types
- Anti Decompiler
- Constant Hider
- AntiTamper
- Anti De4dot


# Usage
Open the compiled application, insert Assembly path press Enter key
->Output: ...-cleaned.exe
